# UN-Refugee-Data-Finder
Exploring data from the United Nations Refugee Data Finder Tool using Python. Data was queried on December 1, 2025. For more information, see: https://www.unhcr.org/refugee-statistics

# Asylum Seekers from the Americas to the United States

## Overview

This project analyzes **asylum seeker flows from the Americas to the United States** using publicly available international data. The goal is to understand **regional and temporal patterns** in asylum demand, as well as how these flows have evolved over time.

The analysis focuses on identifying trends across regions rather than individual countries, allowing for a clearer view of broader migration dynamics affecting the United States.

---

## Data Sources

- **UN Refugee Data Finder**
  - Asylum applications and decisions by country of origin and year
- **United Nations M49 Standard**
  - Regional classification of countries and areas for statistical use

All data are annual and aggregated at the regional level.
